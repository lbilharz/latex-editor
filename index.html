LaTeX Math Editor

Accessible formula input for visually impaired students — and everyone working alongside them.

Math on screen is visual — fractions stack, exponents rise, roots get radical signs. Screen readers and Braille displays can’t navigate that. This editor takes a different approach: type formulas as LaTeX, read them back on a Braille display, and see them rendered as native MathML in real time. Both views stay in sync — one formula, two ways to access it. Source on GitHub
