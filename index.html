How to use

  1. Type LaTeX in the lower field — the formula renders above in real time
  2. Use the toolbar to insert fractions, roots, powers, and more
  3. Click the formula to enter Navigate mode — ← → move between elements, ↑ selects parent
  4. ↓ on a leaf, Enter, or double-click to enter Edit mode — type to insert at the cursor
  5. ↑ to go back to Navigate — type to replace, Backspace to delete
  6. Press Esc to switch to LaTeX editing
About this project

Math on screen is visual — fractions stack, exponents rise, roots get radical signs. Screen readers and Braille displays can't navigate that. This editor takes a different approach: type formulas as LaTeX, read them back on a Braille display, and see them rendered as native MathML in real time. Both views stay in sync — one formula, two ways to access it.
